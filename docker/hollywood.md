#### docker 安装方式
[**bcbcarl/hollywood**](https://github.com/bcbcarl/docker-hollywood/blob/master/README.md)
```powershell
# 查询 hollywood 版本
$ docker search hollywood
# 安装 bcbcarl/hollywood 版本
$ docker pull bcbcarl/hollywood 
# 运行
$ docker run --rm -it bcbcarl/hollywood
```
